# Project #1 - Sonny Says

## User Story
As a user, I want to play a fun and interactive single player game with clear instructions that gives instant feedback on whether I win or lose. 

### Acceptance Criteria:
- Single player game
- Clear instructions
- Interactive
- Win/Loss case
